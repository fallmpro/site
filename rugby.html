Quelques info sur le projet :

- Durée : 3 Mois

- Compétences à acquérir : Django ; Python; Gestion d'API; Backend

Le projet Rugby Tropical Cup comprend un site internet ainsi que deux pages web mobiles. Cette plateforme centralise les informations sur les matchs de rugby et les détails des événements associés. Les utilisateurs munis de billets peuvent les scanner pour accéder instantanément aux informations spécifiques de chaque match, telles que les équipes participantes, les dates et les stades.
